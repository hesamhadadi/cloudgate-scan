🛡️ CloudGate Tunnel

تونل از طریق DNS-over-HTTPS. اگه نت ملی همه چیز رو بسته — این هنوز کار می‌کنه چون فقط از dns.google یا 1.1.1.1 رد می‌شه (که ایران مجبوره بازشون نگه داره).
Server: Frankfurt 🇩🇪 Protocol: dnstt + microsocks

📥 دانلود سریع

📦 دانلود کامل (همه پلتفرم‌ها) — 36MB

شامل Windows + Mac + Linux + Android. مناسب اگه نمی‌دونی کاربر چی داره.

یا فقط برای OS خودت:

🪟 Windows x64 🍎 macOS (Apple Silicon) 🍎 macOS (Intel) 🐧 Linux x64 📱 Android (Termux ARM) 🪟 Windows ARM

اسکریپت‌های راه‌اندازی: start-windows.bat · start-mac.command · start-linux.sh

⚡ راه‌اندازی (۳۰ ثانیه)

1
دانلود زیپ بالا، اکسترکت کن.
2
روی Windows: start-windows.bat دابل‌کلیک.
روی Mac: راست‌کلیک start-mac.command → Open.
روی Android: نصب اپ Termux از F-Droid، فایل‌ها رو کپی به /sdcard/tunnel/، تو Termux بزن bash /sdcard/tunnel/start-linux.sh.
3
تو پنجره می‌بینی: SOCKS5 proxy on 127.0.0.1:7000.
پنجره رو ببند → تونل قطع. باز نگهدار.
4
تلگرام دسکتاپ: Settings → Advanced → Connection type → Use custom proxy
Type: SOCKS5 | Hostname: 127.0.0.1 | Port: 7000
تلگرام اندروید: Settings → Data → Proxy Settings → Add → SOCKS5 → 127.0.0.1 / 7000
برای فایرفاکس و کروم

Firefox: Settings → Network Settings → Manual proxy → SOCKS Host: 127.0.0.1 Port: 7000 → SOCKS v5 → ☑ Proxy DNS when using SOCKS v5

Chrome/Edge: اکستنشن FoxyProxy یا Proxy SwitchyOmega نصب کن، پروفایل SOCKS5 با همون آدرس بساز.

چطوری کار می‌کنه؟ 
[Telegram یا مرورگر]
        ↓ SOCKS5 127.0.0.1:7000
[dnstt-client روی دستگاه]
        ↓ HTTPS port 443 to dns.google
[Google DoH resolver]   ← ایران این رو نمی‌بنده
        ↓ recursive lookup *.t.jizjiz.fun
[Cloudflare nameservers] → delegated to:
[سرور آلمان UDP/53]      ← مسیر Google→DO، نه Iran→DO
        ↓ unwrap encrypted payload
[microsocks → اینترنت آزاد]

چون ترافیک از طریق Google/Cloudflare DoH رد می‌شه (که ایران بازشون نگه می‌داره)، حتی وقتی IP سرور خارجی کاملاً null-routeـه، باز کار می‌کنه.

اگه وصل نشد
  • اسکریپت چند تا resolver رو پشت‌سرهم امتحان می‌کنه. اگه اولی فِیل کرد، صبر کن بعدی رو شروع کنه.
  • روی Mac اگه می‌گه «cannot be opened»: راست‌کلیک → Open → Open.
  • روی Windows اگه Defender هشدار میده: More info → Run anyway. (binary unsigned ولی open-source از dnstt compile شده)
  • سرعت کندـه (50-200KB/s) چون داره از داخل DNS رد می‌شه. برای پیام و چت خوبه، ویدیو نه.
